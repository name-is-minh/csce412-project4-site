# csce412-project4-site

Personal multi-page static website hosted serverlessly on AWS with a fully automated CI/CD deployment pipeline.

**Live Site:** [csce412mhnguyen.me](https://csce412mhnguyen.me)

## Site Structure

- index.html
- about.html
- projects.html
- experience.html
- skills.html
- contact.html
- assets/css/styles.css
- assets/js/main.js
- assets/img/*

## Local Preview

Open index.html directly in browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit:

- <http://localhost:8080>
